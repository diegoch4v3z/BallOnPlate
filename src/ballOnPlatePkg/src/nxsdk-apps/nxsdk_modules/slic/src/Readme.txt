The image data used in images folder : infer_images.binary, infer_labels.binary, train_images.binary and
train_labels.binary are obtained by processing Columbia Object Image Library (COIL-20) dataset authored by
Sameer A. Nene , Shree K. Nayar , Hiroshi Murase (http://www.cs.columbia.edu/CAVE/software/softlib/coil-20.php).
