class nxsdk.arch.n2a.compiler.tracecfggen.tracecfggen.TraceCfgGen¶

A generator for TraceCfg objects. Provides convenience methods to generate trace decay factors, spikeLevel, randomThreshold and expTimeConstant from higher level input parameters.

calcDecay(tau)¶

Calculates trace decay factors from trace decay time constant. Every trace decay factor is given by: decay[i] = exp(-2**i/tau)*(2**W_TRACE_DECAY-1)

Parameters

tau -- (int) Positive trace decay time constant.

calcExpTimeConst(tau)¶

Calculates exponential time constant from trace decay time constant.

Parameters

tau -- (int) Positive trace decay time constant.

calcOptimalSpikeLevel(dtMin, tau)¶

Calculates optimal spikeLevelInt and spikeLevelFrac from minimum expected inter spike interval and trace decay time constant. The spikeLevel will be chosen such that the maximum trace value is 2**W_TRACE-1 as long as the minimum inter spike interval is larger than dtMin otherwise the trace will saturate.

Parameters

dtMin -- (int) Minimum inter spike interval above which trace

does not saturate. :param tau: (int) Positive trace decay time constant. :returns: (<int>, <int>) (spikeLevelInt, spikeLevelFrac)

calcRndThresh(tau)¶

Calculates threshold below which trace is randomly rounded via random decrementing.

Parameters

tau -- (int) Positive trace decay time constant.

calcSpikeLevel(relativeSpikeLevel)¶

Calculates absolute spikeLevelInt and spikeLevelFrac based on a relativeSpikeLevel.

Parameters

relativeSpikeLevel -- (int) A value between 0 and 1 that gets

scaled to 0..(2**W_TRACE-1)

calcTau(decay)¶

Calculates trace decay time constant from trace decay factors. The trace decay time constant is the average of: tau[i] = -2**i/ln(decay[i])

Parameters

decay -- (tuple-6) Trace decay factors.

genTraceCfg(tau, spikeLevelInt, spikeLevelFrac)¶

Generates a TraceCfg object.

Parameters

  • tau -- (int) Positive trace decay time constant.

  • spikeLevelInt -- (int) Integer spikeLevel by which trace

increases upon spike. :param spikeLevelFrac: (int) Fractional spikeLevel by which trace increases upon spike.
