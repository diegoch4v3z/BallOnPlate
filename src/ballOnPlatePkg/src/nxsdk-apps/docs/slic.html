SLIC¶

The Single Layer Image Classifier (SLIC) module provides functions to build one layer neural network for image classification. The images for SLIC can be grayscale images stored as a numpy array. SLIC module provides function to preprocess grayscale images into binary images, which drive linescan engine to generate spikes that are further processed by Loihi. Both the image data and the corresponding labels are needed to train the network.

Jupyter notebook illustrating a Single Layer Image Classifier resides at nxsdk_modules/slic/tutorials

configureCx(compartmentCurrentDecay=410, compartmentVoltageDecay=410, refractoryDelay=2, enableSpikeBackprop=1, enableSpikeBackpropFromSelf=1, vThMant=9000, vMinExp=23, vMaxExp=23, enableHomeostasis=1, minActivity=0, maxActivity=127, homeostasisGain=0, activityImpulse=1, activityTimeConstant=1000000, cxPrototype=None)¶
Parameters

  • compartmentCurrentDecay -- Decay constant by which the compartmentCurrent decays per time step.

  • compartmentVoltageDecay -- Decay constant by which the compartmentVoltage decays per time step.

  • refractoryDelay -- After a compartment spikes it enters a refractory state of duration refractoryDelay during which compartmentVoltage does not accumulate new inputs.

  • enableSpikeBackprop -- Enables spike backpropagation mechanism.

  • enableSpikeBackpropFromSelf -- Enables spike backpropagation when compartment spikes itself.

  • vThMant -- Threshold voltage of the compartment.

  • vMinExp -- vMinExp determines the minimum voltage vMin at which compartmentVoltage saturates.

  • vMaxExp -- vMaxExp determines the minimum voltage vMin at which compartmentVoltage saturates.

  • enableHomeostasis -- enableHomeostasis determines whether vThMant gets updated via activity driven threshold homeostasis.

  • minActivity -- minActivity is the lower threshold below which vThMant gets decreased to increase the compartment activity.

  • maxActivity -- maxActivity is the upper threshold above which vThMant gets increased to reduce the compartment activity

  • homeostasisGain -- homeostasisGain controls how strongly vThMant gets modified by activity leaving the range defined by minActivity, maxActivity.

  • activityImpulse -- activitiyImpulse specifies the amount by which activity increases for each compartment spike.

  • activityTimeConstant -- Decay time constant by which activity decays approximately exponentially over time.

  • cxPrototype -- Compartment Prototype, if provided all compartments created will have this prototype. All the other parameters will be ignored.
