NxNet API¶

NxNet Module

class nxsdk.net.net.NxNet(compilerType=<CompilerType.DEFAULT: 'N2'>)¶

NxNet represents a network of Neurons, Compartments and Connections forming a neural network.

Following is an example to create and run a simple NxNet network connecting two compartment groups with a connection mask:

import nxsdk.api.n2a as nx
import np as np

net = nx.NxNet()

compartmentPrototype = nx.CompartmentPrototype(biasMant=100,
                                               biasExp=6,
                                               vThMant=1000,
                                               functionalState=2,
                                               compartmentVoltageDecay=256)

connectionPrototype = nx.ConnectionPrototype(weight=200)

# Create two compartment groups with the same prototype
compartmentGroup1 = net.createCompartmentGroup(size=10, prototype=compartmentPrototype)
compartmentGroup2 = net.createCompartmentGroup(size=10, prototype=compartmentPrototype)

# Create a connection group to connect the two compartment groups
connectionGroup1 = compartmentGroup1.connect(compartmentGroup2,
                                             connectionPrototype,
                                             connectionMask=np.eye(10))

net.run(100)
net.disconnect()

createCompartmentGroup(name=None, size=0, prototype=None, prototypeMap=None, **kwargs)¶

Creates new compartments according to prototype(s) and groups them in a logical group.

Parameters

  • name (str) -- An optional name for the group. If not provided, a name will be automatically generated.

  • size (int) -- Size of logical group. Default is 0 for an empty group in which case no prototype or prototypeMap is expected.

  • prototype (NodePrototype or list(NodePrototype)) -- A single NodePrototype object or a list of NodePrototype objects. If a single NodePrototype is provided, it is used for all new compartments and no prototypeMap is expected.

  • prototypeMap (list(int)) -- If 'prototypes' is a list of NodePrototypes, 'prototypeMap' must be a list of integers of length 'size'. Each list element prototypeMap[j] must be an index into 'prototypes' and specifies the NodePrototype used for the new compartment 'j' in this group.

  • kwargs -- Pass any of the valid key/value pairs for a Compartment's discrete states.

createConnectionGroup(name=None, src=None, dst=None, prototype=None, prototypeMap=None, connectionMask=None, **kwargs)¶

Creates new connections according to prototype(s) and groups them in a logical group. CompartmentGroup.connect is an alias to this method and allows to connect CompartmentGroups more intuitively according to:

conn = compartmentGroup1.connect(compartmentGroup2, prototype)
Parameters

  • name=None (str) -- A descriptive name for the group.

  • src=None (NodeGroup) -- The source Compartment/NeuronGroup of the ConnectionGroup.

  • dst=None (NodeGroup) -- The destination Compartment/NeuronGroup of the ConnectionGroup.

  • prototype=None -- A single ConnectionPrototype or list of ConnectionPrototype objects. If a single object is provided, it is used for all connections in the group. If a list is provided prototypeMap must be provided as well. :type prototype: NodePrototype or list(NodePrototype)

  • prototypeMap=None -- A 0/1/2D np array where each element must be within 0..len(prototype) which specifies the prototype to use for each connection. A non-2D array is automatically broadcast into a (numDstNodes x numSrcNodes) array.

  • connectionMask (np.array) -- A 0/1/2D np binary np.ndarray or 2D scipy.sparse.spmatrix which specifies which connection exists within the (numDstNodes x numSrcNodes) space. Elements in prototypeMap, weights, delays and tags where connectionMask == 0 will be ignored. A non-2D array is automatically broadcast into a (numDstNodes x numSrcNodes) array.

  • weight (np.array) -- A 0/1/2D np.ndarray or 2D scipy.sparse.spmatrix weight matrix. A non-2D array is automatically broadcast into a (numDstNodes x numSrcNodes) array.

  • delay (np.array) -- A 0/1/2D np.ndarray o2 2D scipy.sparse spmatrix delay matrix. A non-2D array is automatically broadcast into a (numDstNodes x numSrcNodes) array.

  • tag (np.array) -- A 0/1/2D np.ndarray or 2D scipy.sparse.spmatrix tag matrix. A non-2D array is automatically broadcast into a (numDstNodes x numSrcNodes) array.

  • enableLearning (np.array) -- A 0/1/2D np.ndarray or 2D scipy.sparse.spmatrix enableLearning matrix. A non-2D array is automatically broadcast into a (numDstNodes x numSrcNodes) array.

createDVSSpikeGenProcess(xPixel, yPixel, polarity=2, serviceInterval=0)¶

Creates a DVS spike generator for a given image resolution size of xPixel * yPixel times the polarity dimension; where the polarity is the sign of the brightness change for that pixel. Here, the polarity parameter represents the number of dimensions for the polarity - either 1 or 2. The default is 2, which provides the distinction between brightness increasing or decreasing when the pixel value changes.

The DVS spike generator process uses a fixed mapping from the DVS camera to the input axons: -- (x,y,pol) to an index idx=x*(180*2)+y*2+pol, range of idx is 0 to 240x180x2-1 (86399) -- Map the idx sequentially to cores and input axons, i.e. core=idx/1024, axon=idx%1024.

Note

This function is being deprecated. Use the DVS module instead.

Parameters

  • xPixel(int) -- total number of pixels in the xDimension

  • yPixel(int) -- total number of pixels in the yDimension

  • polarity(int) -- number of dimensions of polarity, valid values are 1 or 2

  • serviceInterval(int) -- Not Implemented

Returns

DVS spike generator object

createInteractiveSpikeGenProcess(numPorts)¶

Creates an interactive spike generator with a specified number of ports. Each port can be connected to a single compartment or one or more compartments within a compartment group through a connection prototype. All spikes along that port will be sent to the compartments specified by the connection.

net = nx.NxNet()

cp = nx.CompartmentPrototype(vThMant=10, functionalState=nx.COMPARTMENT_FUNCTIONAL_STATE.IDLE)
cx = net.createCompartment(prototype=cp)
probes = cx.probe([nx.ProbeParameter.SPIKE])

connproto = nx.ConnectionPrototype(weight=255)

sp = net.createSpikeGenProcess(numPorts=1)
sp.connect(cx, prototype=connproto)

# Add spikes to be inserted before invoking run
sp.addSpikes([0], [[1, 2, 3, 4, 5]])

si = net.createInteractiveSpikeGenProcess(numPorts=1)
si.connect(cx, prototype=connproto)

net.runAsync(2000)

# Now send spikes interactively
for i in range(50):
    sleep(0.2)
    si.sendSpikes([1], [1])

net.disconnect()
Parameters

numPorts(int) -- Number of input ports for spike injection

Returns

Interactive spike generator object

createLearningRule(**kwargs)¶

Creates a new LearningRule. The configuration of a learning rule requires the equations that govern the dynamics of the weight (dw), delay (dd) and tag (dt) state variables as well as the trace parameters (impulse, timeConstant) for up to two preTraces (x1, x2), up to three post traces (y1, y2, y3) and one reward trace (r1). And the duration of the learning epoch (tEpoch).

All impulse values must be within the range 0..127 while all timeConstant must be positive integers. (See Specifying a Learning Rule class for further details.)

Parameters

  • dw (string) -- Unique id of LearningRule assigned by NxNet.

  • dw -- Equation that describes dynamics of synaptic weight.

  • dd (string) -- Equation that describes dynamics of synaptic delay.

  • dt (string) -- Equation that describes dynamics of synaptic tag.

  • x1Impulse (int) -- Impulse by which preTrace increases upon each pre-synaptic spike.

  • x1TimeConstant (int) -- Time constant by which preTrace decays exponentially over time.

  • x2Impulse (int) -- Impulse by which preTrace increases upon each pre-synaptic spike.

  • x2TimeConstant (int) -- Time constant by which preTrace decays exponentially over time.

  • y1Impulse (int) -- Impulse by which postTrace increases upon each post-synaptic spike.

  • y1TimeConstant (int) -- Time constant by which postTrace decays exponentially over time.

  • y2Impulse (int) -- Impulse by which postTrace increases upon each post-synaptic spike.

  • y2TimeConstant (int) -- Time constant by which postTrace decays exponentially over time.

  • y3Impulse (int) -- Impulse by which postTrace increases upon each :param int y3TimeConstant: Time constant by which postTrace decays exponentially over time.

  • r1Impulse (int) -- Impulse by which rewardTrace increases upon each reward spike.

  • r1TimeConstant (int) -- Time constant by which rewardTrace decays exponentially over time.

  • tEpoch (int) -- Duration of learning epoch. tEpoch of all learning rules must be identical. tEpoch must be in the range 0..64. It is recommended to use powers of 2 for efficiency.

  • combineLiterals (bool) -- If True, combines all literals in equation into a single [+,-]mantissa*2^[+,-]exp representation.

  • printDebug (bool) -- If true, prints compilation process of learning rule.
