Working with Monitor¶

The monitor helps you to define and organize your probes. More information on probes can be found in Working with Probes

Here's a simple code snippet you will find in most of the tutorials:

uProbe = mon.probe(n2Core.cxState, [0], 'u')[0]

fig = plt.figure(1001)

plt.subplot(1, 2, 1)
uProbe.plot()
plt.title('u')
class nxsdk.graph.monitor.monitor.Monitor(nxGraph)¶

A Monitor is used to monitor the temporal evolution of Graph state. A Monitor contains Probes that describe what to monitor and TimeSeries that contain the temporal evolution of Graph state. A user makes calls to Monitor.probe(..) and provides information on what to monitor in a NodeSet. Each NodeSet in turn is responsible for generating an appropriate Probe for probing its state. All probes are eventually processed by Compiler to map high level state addresses to N2Driver specific state addresses. N2Driver uses Probe information during simulation to produce TimeSeries data. Finally, TimeSeries data can be retrieved for further data analysis.

Creates a TimeSeries for every Probe.

probe(nodeSet, nodeIds, fieldName, probeCondition=None, **kwargs)¶

Creates new probes for all states specified.

Parameters

  • nodeSet -- The nodeSet for which to create probes.

  • nodeIds -- List of integer ids of nodes to be probed.

  • fieldName -- Name of field to monitor.

  • probeCondition -- (ProbeCondition) Describes the condition under which data is monitored.

Kwargs

Additional key/pairs that ProbeGenerator may need.

probesToNumpy(probes, tMin=None, tMax=None)¶

Returns timeSeries.data as numpy array within time window defined by tMin and tMax.

Parameters

timeSeries (list) -- List of timeSeries objects. All timeSeries

objects must be associated with same probeCondition. :param int tMin: Beginning of time window. If None, then it will be ignored. :param int tMax: End of time window. If None, then it will be ignored. :return: (numSteps,) timeSteps vector. :rtype: numpy.ndarray :return: (numSteps, len(timeSeries)) matrix of time series data. :rtype: numpy.ndarray

Sets a new defaultProbeCondition that is used when no probeCondition is provided to Monitor.probe method.
