addProcess(process: nxsdk.composable.interfaces.process.Process)¶

Adds a process to this composable which provides the ability to execute user code on lmt/host. Execution of processes (on lmt/host) within composables or among composables can be ordered using the executeAfter or executeBefore API associated with Process object.

Only EmbeddedSnip processes are currently supported by the composable

During the model pipeline compilation, all processes are grouped by chipId, lmtId and phase. For each group, a wrapper with the aggregated snip code honoring any ordering is generated and attached to the board.

See Process for additional information

process -- (Process) to be added to this composable
