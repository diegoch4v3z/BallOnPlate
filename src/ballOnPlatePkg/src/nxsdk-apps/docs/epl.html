EPL¶

The EPL module provides functionality to setup, train and test a neuromorphic algorithm for one-shot pattern learning inspired by the External Plexiform Layer (EPL) network of the mammalian olfactory bulb. While the algorithm implemented by the network learns to recognize "odors" (more precisely, using odor chemosensor array data), it can be used to learn any other patterns as the EPL network basically implements a spatio-temporal attractor (STA) network. After the EPL/STA network "learns" an odor/pattern, even if a noise corrupted version of the learned pattern (test sample) is presented, the network activity settles down into the attractor state of the learned pattern.

The module provides a EPLMultiPatternLearning class to setup, train, test and evaluate the performance of a one-shot learning algorithm.

Jupyter notebook illustrating a Multi Pattern One-shot Learning Using EPL Network resides at nxsdk_modules/epl/tutorials

Parameters For EPL Network¶

class nxsdk_modules.epl.src.multi_pattern_learning.epl_parameters.ParamemtersForEPL¶

Sets the default values for the parameters to EPL network. The values of the parameter are validated every time they are set.

Parameters

  • numPatterns (int) -- number of patterns to be learned by the EPL network

  • numColumns (int) -- number of columns in the network

  • numMCsPerColumn (int) -- number of MCs per column

  • numGCsPerPatternPerColumn (int) -- number of GCs per column for each different pattern

  • numDelaysMCToGC (int) -- number of different MC->GC connnection delays

  • minDelaysMCToGC (int) -- value of the smallest MC->GC delay

  • gammaCycleDuration (int) -- number of algorithmic timesteps in a gamma cycle

  • connProbMCToGC (float) -- MC->GC connection probability

  • useRandomSeed (bool) -- used to set if we want to use a random seed

  • randomGenSeed (int) -- the value of random seed

  • numGammaCyclesTrain (int) -- number of gamma cycles to train the network

  • numGammaCyclesTest (int) -- number of gamma cycles to test the network

  • numGammaCyclesIdle (int) -- number of gamma cycles when no inputs are presented

  • useLMTSpikeCounters (bool) -- set to true if we want to use SNIP based spike counter instead of spike probes

  • logSNIPs (bool) -- used to turn SNIP logging on and off

  • executionTimeProbe (bool) -- set to enable/disable execution time probe
