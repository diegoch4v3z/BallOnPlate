Axon Compiler¶

class nxsdk.arch.n2a.compiler.axongen.axon_compiler.N2AxonCompiler¶
compile(nxCore, logicalToPhysicalChipIdTable, chipGen)¶

Converts the axons on the provided core into axonMap and axonCfg configurations.

Parameters

  • nxCore -- The logical core on which to configure output axons

  • logicalToPhysicalChipIdTable (list) -- Index k into the list provides the physical chip ID for logical chip ID k

  • chipGen (ChipGen) -- Specifies the chip generation being used

Returns

None

class nxsdk.arch.n2a.compiler.axongen.axon_compiler.PopulationValidator(popId)¶

Validates assumptions regarding compartments and axons within a population. Class to track relative compartment index (atom) values as well as destination synapse map IDs for a given population. Uses the information to ensure that processed output axons satisfy requirements.

addAxon(axon)¶

Adds the information for the output axon into internal data structures.

Parameters

axon (OutputAxon) -- The output axon

Returns

None

getCompartments()¶

Return the set of source compartments in this population.

Returns

Set containing cx IDs

validate()¶

Run validation to ensure that all compartments have a consistent set of destinations. For example, if a compartment goes to dest X and Y, then all compartments must go to dest X and Y.

Returns

None
