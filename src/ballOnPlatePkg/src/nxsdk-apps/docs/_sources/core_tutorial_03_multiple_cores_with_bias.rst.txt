NxCore: Multiple Cores with Bias
================================

Motivation
----------
The motivation for this tutorial is to show you how to connect compartments across multiple cores, sending spikes from a single source compartment to compartments in 3 other cores. Using the function createDiscreteAxon greatly simplifies the wiring between the compartments.
