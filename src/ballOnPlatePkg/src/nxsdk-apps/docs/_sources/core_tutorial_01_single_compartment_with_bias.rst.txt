NxCore: Single Compartment Tutorial
===================================

Motivation
----------
The motivation for this tutorial is to provide you an introduction to the configuration of a single compartment neuron, driven with bias only. Use this tutorial to try out a number of different v threshold values and bias exponent/mantissa values.
