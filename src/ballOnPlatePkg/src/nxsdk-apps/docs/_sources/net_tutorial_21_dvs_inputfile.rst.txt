NxNet: DVS Inputfile
=======================

Motivation
----------
This tutorial creates a DVS network and loads a pre-recorded jAER video file (.aedat) to inject the spikes into the configured DVS network. It also demonstrates how to save
the state of the board and reload it to avoid the compilation time.
