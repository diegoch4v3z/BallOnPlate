NxCore: Simple EI Network Tutorial
==================================

Motivation
----------
The motivation for this tutorial is to provide you an introduction to configuring a single compartment neuron, driven with bias only and connected to two other neurons - one via an inhibitory synapse and one via an excitatory synapse.
