class nxsdk.net.groups.CompartmentGroup(parent, groupId, name)¶

Represents a logical group of compartments.

addCompartments(compartments)¶

Adds new compartments to logical group.

Parameters

compartments -- List of Compartment objects.

addCompartmentsById(compartments)¶

Adds new compartments to logical group. Intended for use with NxSDK internal classes, e.g. to efficiently construct compartment groups. This method differs from addCompartments in the following two ways: 1. This method allows for adding compartments by compartment node ID 2. This method does not verify compartments are in the same network.

Parameters

compartments -- List of Compartment objects or compartment ids.

connect(dstGrp=None, prototype=None, prototypeMap=None, connectionMask=None, name=None, sharedConnGrp=None, synapseSharingOnly=False, **kwargs)¶

Connects CompartmentGroup to destination group which can be another CompartmentGroup.

Parameters

  • dstGrp=None -- The destination Compartment/Receiver of the ConnectionGroup.

  • prototype=None -- A single ConnectionPrototype or list of ConnectionPrototype objects. If a single object is provided, it is used for all connections in the group. If a list is provided prototypeMap must be provided as well.

  • prototypeMap=None -- A 0/1/2D numpy array where each element must be within 0..len(prototype) which specifies the prototype to use for each connection. A non-2D array is automatically broadcast into a (numDstNodes x numDstNodes) array.

  • connectionMask -- A 0/1/2D numpy binary numpy array which specifies which connection exists within the (numDstNodes x numDstNodes) space. Elements in prototypeMap, weights, delays and tags where connectionMask == 0 will be ignored. A non-2D array is automatically broadcast into a (numDstNodes x numDstNodes) array.

  • name=None -- (string) A descriptive name for the group.

  • sharedConnGrp (ConnectionGroup) -- The connection group whose connections are to be shared.

  • synapseSharingOnly (bool) -- True to share synapses only, False to share both axons and synapses.

  • weight -- A 0/1/2D numpy array weight matrix. A non-2D array is automatically broadcast into a (numDstNodes x numDstNodes) array.

  • delay -- A 0/1/2D numpy array delay matrix. A non-2D array is automatically broadcast into a (numDstNodes x numDstNodes) array.

  • tag -- A 0/1/2D numpy array tag matrix. A non-2D array is automatically broadcast into a (numDstNodes x numDstNodes) array.

  • enableLearning -- A 0/1/2D numpy array enableLearning matrix. A non-2D array is automatically broadcast into a (numDstNodes x numDstNodes) array.

getState(state, ids=None)¶

Returns a numpy array view of the discrete state for all ids specified.

Parameters

  • (str) (state) -- Name of discrete state of which data is requested.

  • (sequence) (ids) -- Sequence of relative compartment indices with respect to ComaprtmentGroup. Default is None in which case all states of CompartmentGroup are returned.

Returns

numpy array

probe(cxParametersToProbe, probeConditions=None)¶

Creates a probe for each compartment in the compartment group and returns all probes of the same type as ProbeSets.

Parameters

  • cxParametersToProbe -- The list of compartment state parameters (ex: voltage, current, etc.) to probe.

  • probeConditions -- A list of probeConditions. Each probeCondition will be applied to the corresponding probe specified in the cxParametersToProbe list.

Returns

A tuple of ProbeSet objects. A ProbeSet combines a set of probes of the same type and allows to plot all timeSeries via ProbeSet.plot().

Raises

ValueError -- If there is mismatch between the probe parameters and probe conditions.

setState(state, vals, ids=None)¶

Sets discrete state of a CompartmentGroup to vals for all ids specified.

Parameters

  • (str) (state) -- Name of discrete state to configure.

  • vals -- numpy array of values to set. Must be of a type that can be assigned to a numpy array like numpyArray[ids] = vals.

  • (sequence) (ids) -- Sequence of relative compartment indices with respect to CompartmentGroup. Default is None in which case <vals> is assigned to entire array like numpyArray[:]=vals.

Returns

numpy array
